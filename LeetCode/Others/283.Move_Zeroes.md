# LeetCode
- Move Zeroes
  - [URL](https://leetcode.com/problems/move-zeroes/)   

### 문제 이해하기
nums을 입력받는다.

배열을 순회하면서 모든 0을 배열의 뒤로 보낸다.

새로 배열을 할당하거나 복사하거나 하지 않고 순수하게 nums에서 바꿔서 진행한다.   

### 문제 접근 방법
투포인터를 사용해서 푼다. 

start, end 를 지정한다. 

1. nums[start] 가 0 일 때 nums[end] 가 0이 아닌 숫자가 나올 때까지 돌린다. 
2. nums[start]와 nums[end] 를 바꾼다. 

### 구현 배경 지식
